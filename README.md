## 📌 About The Project

A simple yet engaging **Number Guessing Game** built using Python. The computer randomly picks a secret number between **1 and 100**, and your goal is to guess it in as few attempts as possible! The game gives you hints — *Too High* or *Too Low* — to guide you toward the correct answer.

---

## ✨ Features

- 🎲 Randomly generates a secret number between **1 and 100**
- 💬 Real-time feedback: **Too High / Too Low / Correct!**
- 🔢 Tracks the number of **attempts** taken
- ✅ Input validation — handles non-numeric and out-of-range inputs gracefully
- ⚡ Lightweight & runs directly in the terminal

## 🚀 Getting Started

### Prerequisites

Make sure you have **Python 3.x** installed on your system.

```bash
python --version
```

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/codebyavneesh/number-guessing-game.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd number-guessing-game
   ```

3. **Run the game**
   ```bash
   python number_guessing_game.py
   ```

---

## 🎮 How To Play

1. Run the script in your terminal
2. The computer will secretly pick a number between **1 and 100**
3. Enter your guess when prompted
4. The game will tell you if your guess is **Too High** or **Too Low**
5. Keep guessing until you find the correct number
6. Try to beat your high score by guessing in fewer attempts! 🏆
